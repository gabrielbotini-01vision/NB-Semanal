Design System · Aplicações internas
Um sistema, uma linguagem de produto.

Os princípios visuais dos slides Hotmart traduzidos para UI de app: laranja em evidência, superfícies planas, elementos independentes e zero gradiente. Componentes prontos para colar nos seus frontends.

Fundamentos

Cores

Laranja é a assinatura — deve carregar a hierarquia de ação e destaque em qualquer tela. Preto e off-white sustentam estrutura e superfície. Terciárias entram só em telas expressivas (dashboards, celebração, gamificação).

Diretrizes

Padrões de uso

Os mesmos princípios dos slides, aplicados à interface. Ouse em um único ponto por tela; mantenha o resto disciplinado.

Faça
  • Laranja como cor de ação e de destaque numérico
  • Superfícies planas, separadas por borda e contraste
  • Accent bars finas como assinatura (topo/lateral)
  • Corpo alinhado à esquerda; centralizar só títulos
  • Ícones geométricos em laranja, preto, branco ou off-white
  • No máximo 4 blocos de informação por bloco lógico
Evite
  • Gradientes de qualquer tipo
  • Sombras decorativas (só funcional, < 0,12)
  • Linha decorativa sob títulos
  • Barras coloridas full-width em header/footer
  • Texto de corpo em cinza ou cores terciárias
  • Agrupar texto dentro da forma — mantenha camadas separadas
Implementação

Tokens em CSS

Cole no :root dos seus frontends (GitHub Pages, N8N dashboards). Todos os componentes acima derivam destas variáveis.

tokens.css
/* Cores primárias */
--orange:#FF4000;   --black:#0D0D0D;   --off-white:#F5F3EF;
/* Secundárias */
--gray:#C3BFB8;  --white:#FFFFFF;  --yellow:#FFD450;  --maroon:#3E0F1D;
/* Terciárias (expressivas) */
--lemon:#6DFF96;  --purple:#B48EFF;  --blue:#44D0FF;

/* Semânticas */
--bg:var(--off-white);      --surface:var(--white);
--text:var(--black);        --text-muted:rgba(13,13,13,.6);
--accent:var(--orange);     --border:#E2DED7;

/* Tipografia */
--font-sans:"DM Sans",Arial,sans-serif;

/* Espaçamento · base 4 */
--sp-1:4px; --sp-2:8px; --sp-3:12px; --sp-4:16px;
--sp-5:24px; --sp-6:32px; --sp-7:48px; --sp-8:64px;

/* Raio */
--r-sm:6px; --r-md:10px; --r-lg:16px; --r-pill:999px;
